Cowry Protocol

A Decentralized Fiat Backed Stablecoin

Introduction

Cowry protocol is a new type of anchor build on a distributed Network of Independent Fiat Providers(IFP). Cowry Protocol is the issuer of a new type of stablecoin that is decentralized, Backed by Fiat and controlled by a group of enthusiasts in an open, collaborative and transparent manner, the goal is to provide a stablecoin that combines the stability and trust of fiat plus the decentralized and security of blockchain to a single stablecoin, enjoying the benefit of both worlds. The goal of this design is not to be a Central Bank, the goal is to manage, the issuance of a new type of stablecoin that is decentralized, backed by physical fiat held in a distributed account and managed by a group of the protocol enthusiast through a DAO, in a transparent way using Blockchain technology. The goal is to create a truly peer-2-peer stablecoin that is censorship resistant, backed by Fiat, available for Everyone, convenient to use, etc. APIs are publicly available for anyone/businesses to integrate with and use provided they comply with the rules that governed the protocol as set by the community DAO from time to time. The most effective way to grow the stablecoin and make it censorship-resistant is by businesses joining the list of Independent Fiat Providers(IFP) making it more robust. This design allows Independent Fiat Providers to hold Fiat in their physical bank account rather than the common way for Fiat backed Stablecoin which allows a single organization to hold fiat in a single Bank Account.

This specific Protocol will only support Naira Backed stablecoin at the moment, we plan to add more once community is comfortable with the design

API Details:

Accept Parameters

The api list below are publicly available for anyone to use, we plan to support sep6 for easy withdrawals and deposit

The API will respond to the following 'Accept' values with appropriate content.

  • text/plain - Content will be returned as a plain string.
  • application/json - Content will be returned as a JSON object.
